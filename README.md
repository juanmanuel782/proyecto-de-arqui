# proyecto-de-arqui
Proyecto de arqui
El diseño de microprocesadores modernos busca maximizar el rendimiento mediante el paralelismo a nivel de instrucción (ILP). La arquitectura MIPS (Microprocessor without Interlocked Pipeline Stages) es el estándar académico e industrial para ilustrar cómo la segmentación (pipelining) permite incrementar el throughput de un sistema, ejecutando múltiples instrucciones simultáneamente en diferentes fases de completitud.
Objetivos
Construir y verificar un microprocesador basado en la arquitectura MIPS de 32 bits con segmentación de 5 etapas, integrando una herramienta de traducción de instrucciones (Assembly a Binario) para validar su funcionamiento mediante la ejecución de algoritmos que involucren aritmética, lógica, control de flujo y acceso a memoria.
